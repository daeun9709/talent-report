# 신규입사자 인재리포트

피플팀 내부용 라이브 인재리포트 시스템입니다.

## 파일 구조

| 파일 | 설명 |
|---|---|
| `index.html` | 📊 인재리포트 메인 (라이브 대시보드) |
| `ai-tool.html` | 🤖 AI 태그 추출 + 주목 인재 선정 도구 |
| `data.csv` | 📋 구글 시트 초기 데이터 템플릿 |

## 사용 방법

### 1. 구글 시트 연결
1. `data.csv` 파일을 구글 시트로 가져오기
2. 파일 → 공유 → 웹에 게시 → CSV 형식 선택 → 게시
3. `index.html` 열기 → 상단 설정에서 URL 입력 → 연결하기

### 2. 새 직원 추가
1. `ai-tool.html` 에서 소프트랜딩 문서 업로드 → AI 태그 추출
2. "한 줄 복사" → 구글 시트 새 행에 붙여넣기
3. `index.html` 에서 새로고침 → 자동 반영

### 3. 주목 인재 선정
1. `ai-tool.html` → "주목 인재 선정" 탭
2. 구글 시트 전체 복사 → 붙여넣기 → 선정
