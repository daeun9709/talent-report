📋 구글 시트 연결
1. 다운받은 인재리포트_데이터.csv 파일을 구글 시트로 가져오기
2. 구글 시트 상단 메뉴 → 파일 → 공유 → 웹에 게시
3. 게시 형식을 쉼표로 구분된 값(.csv) 선택 → 게시 클릭
4. 생성된 URL 복사 → 아래에 붙여넣기 → 연결하기 클릭
⚠ URL은 .../pub?gid=0&single=true&output=csv 형식이어야 해요
데이터를 불러오지 못했어요 URL을 확인하거나 시트가 "웹에 게시" 상태인지 확인해주세요.
